# Sahyog_Bharat
Sahyog Bharat is a digital initiative designed to bridge the gap between government schemes and the citizens who need them the most. It simplifies access to welfare programs by providing clear, multilingual, and verified information about central and state government schemes — all in one platform


├── index.html # Main homepage / landing page
├── chatbot.html # Chatbot interface
├── ngo.html # NGO information page
├── schemes.html # Schemes listing / awareness page
├── jszip.zip # JavaScript assets (compressed)
├── csszip.zip # CSS stylesheets (compressed)
├── logzip.zip # Logs / supporting files (compressed)

GETING STARTED :---------

Clone the Repository git clone https://github.com/mmeghashree456/SahyogBharat.git cd SahyogBharat

Extract Assets

Unzip the following files into respective folders:

jszip.zip → /js

csszip.zip → /css

logzip.zip → /logs (if required)


Your folder should look like this after extraction:

Slidebar/ ├── index.html ├── chatbot.html ├── ngo.html ├── schemes.html ├── js/ │ └── ...JavaScript files ├── css/ │ └── ...CSS files ├── logs/ └── ...log files

Run the Project
Simply open index.html in your browser.
